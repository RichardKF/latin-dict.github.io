A Latin Dictionary, Lewis & Short (1879)

Well known classic Latin–English dictionary. See also its simplified version “An Elementary Latin Dictionary” (Lewis, 1890).

On-line versions of this dicitonary are available in several places:

  • Latinitium
  • Alatius

Download

Download files compiled for dictionary shells. Format Slob is recommended for GoldenDict, read documentation about other dictionary shells.

Version in DSL (ABBYY Lingvo) format can be found in the Dadako’ Knigohran.
