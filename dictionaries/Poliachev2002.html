Большой русско-латинский словарь, Poliachev (2002)

Latin: Vocabularium Russico-Latinum Magnum.

Russian-Latin dictionary containing contemporary Latin lexis was compiled by Poliachev M. P. (Поляшев Михаил Петрович, 1951-2005).

Download

Current version is v1.1 (February 20, 2020). Format Slob is recommended for GoldenDict, read documentation about other dictionary shells. Files in ABBYY Lingvo (DSL) and Pop-Up Dictionary formats can be taken from Lingua Latina Aeterna.

Source files are in the github repository.
